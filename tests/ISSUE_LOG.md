## ❌ 问题清单

### 问题 #1: 移动后行动点未正确减少

**严重程度**: 中等
**状态**: 🟡 需要确认

**问题描述**:
测试用例"移动后行动点-1"失败。测试代码：
```javascript
await t.initGame();  // 重新初始化
await t.page.evaluate(() => handleCityClick(0));
const ap = await t.page.evaluate(() => G.ap);
if (ap !== 0) throw `ap=${ap}`;
```

**测试结果**: `ap=undefined`

**可能原因**:
1. `initGame()` 可能没有正确重置游戏状态
2. `handleCityClick` 在某些条件下未执行
3. G对象在initGame时被重新定义

**验证方法**:
```javascript
// 在浏览器控制台中手动测试
handleCityClick(0);
console.log(G.ap);  // 应该是0
```

**修复建议**:
检查`initGame()`函数是否正确重置了`G.ap`，或者测试时不需要调用`initGame()`。
