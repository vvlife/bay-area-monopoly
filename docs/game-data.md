## 三、游戏机制

### 行动点系统
- 每回合 1 个行动点
- 可选择：移动 / 建工业 / 结束回合
- 行动点用完自动结束回合

### 回合流程
1. 移动到相邻城市
2. 建设工业设施
3. 收取过路费/投资收益
4. 年代事件触发

### 获胜条件
- 成为最富有的玩家
- 控制最多城市
- 资产最高者获胜
